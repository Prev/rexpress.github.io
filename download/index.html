Test regex in fully isolated environment

You can use our app to test regular expressions.
Cause all module and environments run in docker, testing process never effect to your developing environment.

We support basic environments like python, java, nodejs, and php.
And besides, we support regular expressions in special modules like flume, hive, and nginx.
